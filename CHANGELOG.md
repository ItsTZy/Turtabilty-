# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-15

### Added
- Durability-based armor color system for Fabric `1.21.8`.
- Section-based color configuration.
- Single base-color mode with light-to-dark scaling.
- Transition modes: `STEP`, `SMOOTH`, `FADE`.
- Toggle for coloring at full durability.
- Material enable/disable toggles.
- Mod Menu + Cloth Config integration.
- Turt-themed easter-egg color behavior for custom item names.

### Changed
- Mod display name standardized to `Turtabilty`.
- Artifact base name standardized to `turtabilty`.
- Config file path standardized to `config/turtabilty.json`.

### Compatibility
- Legacy config file `config/turtaiblty.json` is automatically migrated.
